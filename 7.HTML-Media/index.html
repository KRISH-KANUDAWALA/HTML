Audio in HTML

The controls attribute is a boolean attribute.

When present, it specifies that audio controls should be displayed.

Audio controls includes :-

  • Play
  • Pause
  • Seeking
  • Volume

Video in HTML

The <video> tag is used to embed video content in a document, such as a movie clip or other video streams.

The <video> tag contains one or more <source> tags with different video sources.

Optional Attributes Are :-

  • Autoplay
  • Controls
  • Height
  • Muted
  • Poster
  • Preload
  • Src
  • Width
